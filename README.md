# CS 330 Portfolio Reflection

## How do I approach designing software?

When designing software, I start by breaking the end goal into manageable pieces. For this scene, I analyzed the reference photo and identified which parts—like the table or the coffee mug—could be built from basic shapes, allowing me to plan the architecture before diving into the code.

This project sharpened my spatial thinking. I learned how scaling, rotation, lighting, and textures must coordinate to create a cohesive environment. By following an iterative process—building, testing, and refining one object at a time—I was able to match the reference more accurately.

These tactics are directly applicable to any large-scale engineering or programming project. Breaking down complexity, planning early, and prioritizing frequent testing are strategies I’ll continue to use in my future work.

## How do I approach developing programs?

I took a modular approach to this project, starting with a stable OpenGL base and layering in features like lighting and camera controls one by one. By building complex items—like the mug and salt shaker—out of simple geometric shapes, I learned how much small iterations in scale and rotation matter for realism.

As I grew more comfortable with the code structure and texture mapping, the workflow became more intuitive. By the final milestone, I could much more easily translate the math in my code into the 3D visuals on the screen.

## How can computer science help me in reaching my goals?

Computer science provides the toolkit I need to build simulations and interactive systems that turn abstract concepts into clear visuals. Mastering computational graphics is a bridge to my future goals in engineering and 3D modeling, as it teaches me exactly how objects are represented and manipulated in a digital space.

Professionally, these skills are essential for creating prototypes, technical simulations, and visual tools. This project strengthened my ability to bridge the gap between complex code and real-world visual representation.
